## Overview

This document will cover how to configure and manage AI agents in LeCoder AI, including Claude Code, Gemini CLI, and Codex CLI.

## Supported Agents

### 1. Claude Code
- **Provider**: Anthropic
- **Models**: Claude 3.5 Sonnet, Claude 3 Opus
- **Capabilities**: Code generation, debugging, refactoring
- **Configuration**: API key required

### 2. Gemini CLI
- **Provider**: Google
- **Models**: Gemini 2.0 Pro, Gemini 1.5 Pro
- **Capabilities**: Code generation, analysis, multi-modal
- **Configuration**: API key required

### 3. Codex CLI
- **Provider**: OpenAI
- **Models**: GPT-4, GPT-4 Turbo
- **Capabilities**: Code generation, completion, chat
- **Configuration**: API key required
