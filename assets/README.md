# LeCoder AI Logo Placeholder

This directory will contain LeCoder AI branding assets:

- `lecoder-icon.png` - Extension icon (128x128 or 256x256)
- `lecoder-logo.svg` - Vector logo for documentation
- `lecoder-banner.png` - Marketplace banner (optional)

## Temporary Assets

Until proper branding assets are created, the extension uses placeholder icons from the forked Kilocode repository. These should be replaced with LeCoder-specific branding.

## Asset Requirements

### Extension Icon (`lecoder-icon.png`)
- Format: PNG
- Size: 128x128px or 256x256px
- Background: Transparent or solid color
- Style: Simple, recognizable at small sizes

### Logo (`lecoder-logo.svg`)
- Format: SVG (vector)
- Use for: README, documentation, website
- Style: Scalable, works in light and dark themes

### Banner (`lecoder-banner.png`)
- Format: PNG
- Size: 980x200px (VS Code Marketplace standard)
- Use for: Marketplace gallery
- Content: Logo + tagline + gradient background

## Design Guidelines

- **Colors**: Blue/Purple gradient theme (to be defined)
- **Typography**: Modern, clean sans-serif
- **Tagline**: "Less Code, More Creation"
- **Theme**: Research, AI, Collaboration
